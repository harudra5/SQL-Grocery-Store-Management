# Grocery Store Management System

This project simulates a real-world grocery store database using SQL. It includes database design, table relationships, sample data generation, and business-focused analytical queries. The project covers sales analysis, customer insights, product performance, supplier management, employee performance, and inventory tracking.

## Skills Demonstrated
- SQL
- Joins
- Aggregate Functions
- Subqueries
- CTEs
- Window Functions
- Database Design
- Business Analysis
- Data Analytics

## Database Modules

- Products
- Categories
- Suppliers
- Employees
- Orders
- Order Details
